## Frontend (React + Tailwind)

Responsive farmer-friendly dashboard built with React, Tailwind CSS, Recharts, and Framer Motion.

### Install and run
```powershell
cd frontend
npm install
npm run dev
```

### Build for production
```powershell
npm run build
npm run preview
```

### Key folders
- `src/components/layout`: sidebar and navbar
- `src/components/filters`: crop and date controls
- `src/components/charts`: trend/prediction charts
- `src/components/cards`: forecast cards
- `src/pages`: Home, Predictions, Trends, About screens
- `src/services`: API service layer (mock for now, real APIs in Step 7)
